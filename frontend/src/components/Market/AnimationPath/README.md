# 动画路径编辑器

这是一个用于创建和编辑动画路径的组件。它允许用户通过可视化界面设计对象的移动轨迹。

## 功能特性

- 可视化路径编辑
- 实时动画预览
- 动画参数配置(持续时间、循环、缓动等)
- 网格辅助和参考点
- 撤销/重做支持
- 导入/导出路径数据
- 错误边界处理

## 组件结构

- `PathEditor` - 主编辑器组件
- `PathPreview` - 路径预览和编辑画布
- `PathControls` - 动画控制面板(播放、暂停等)
- `PathSettings` - 动画和约束参数设置
- `PathToolbar` - 工具栏(保存、导入导出等)
- `ErrorBoundary` - 错误处理边界
