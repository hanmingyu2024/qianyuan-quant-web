# qianyuan-quant-enterprise

# 千猿量化交易系统

一个功能完整的量化交易系统，支持多策略、实时数据处理、风险控制等功能。

## 功能特点

- 多策略支持
- 实时市场数据处理
- 自动化交易执行
- 风险控制和监控
- 性能分析和报告
- WebSocket实时数据推送
- RESTful API接口

## 系统架构

- 后端：Python (FastAPI)
- 数据库：PostgreSQL
- 缓存：Redis
- 消息队列：Redis Pub/Sub
- 监控：Prometheus + Grafana
- 容器化：Docker
